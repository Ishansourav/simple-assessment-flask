Welcome to Simple Assessment

Built with Flask 3.0.2, this app enables CSV/JSON file uploads, dynamic table rendering, and real-time calculations using Pandas.

CSV File

CSV (Comma-Separated Values) is a simple, text-based format for storing tabular data. It’s fast, human-readable, and widely used in Excel, databases, and data science workflows.

JSon File

JSON (JavaScript Object Notation) is a lightweight format for structured data—ideal for APIs, web apps, and data exchange across modern programming languages.

Tech-stack & Features

Built with Flask 3.0.2, this app handles CSV/JSON parsing using Pandas, renders dynamic tables, and performs calculations (Alpha, Beta, Charlie). Deployed on Heroku with Gunicorn, using HTML + Jinja2 and version-controlled on GitHub.
